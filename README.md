# 📚 Library Management System (Python)

A simple **Library Management System** built using **Python** and **datetime module**.  
This project allows users to manage library books, including adding, searching, issuing, returning, and deleting books — all through a command-line interface.

---

## 🧩 Features

✅ **Add New Book** – Add book details such as title, author, year, and number of copies.  
✅ **Display All Books** – View the complete list of available and issued books.  
✅ **Search Book** – Search for a specific book by its title.  
✅ **Issue Book** – Issue a book and automatically set its return date (7 days from the issue date).  
✅ **Return Book** – Return an issued book and calculate fines if returned late (₹10 per day).  
✅ **Delete Book** – Remove a book from the library.  
✅ **Status Update** – Automatically updates the book status as **Available** or **Out of Stock**.  

---

## 🛠️ Technologies Used

- **Language:** Python 3  
- **Libraries:** `datetime` (built-in module)

---

## 🧮 Fine Policy

- Each book must be returned **within 7 days**.  
- A fine of **₹10 per day** is charged for late returns.
